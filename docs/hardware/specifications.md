# PaperColor 硬件规格

| 规格 | 参数 |
|------|------|
| **SoC** | ESP32-S3R8，双核 Xtensa LX7 @ 240MHz |
| **Flash** | 16MB |
| **PSRAM** | 8MB (Octal，40MHz，CLK=G30，CS=G26) |
| **Wi-Fi** | 2.4GHz，802.11 b/g/n |
| **屏幕** | 4" E-Paper E6 Full-Color (ED2208-DOA / EL040EF1)，400×600 |
| **输入电源** | USB Type-C DC 5V |
| **电池** | 1250mAh 锂电池 |
| **音频编解码** | ES8311 |
| **麦克风** | MEMS + ES7210 ADC，集成 AEC 回声消除 |
| **扬声器** | 1W @ 8Ω 2520 扬声器 + AW8737A 功放 |
| **温湿度** | SHT40 (±1.5% RH, ±0.2°C) |
| **RTC** | RX8130CE |
| **存储** | microSD 卡槽 (SPI 模式) |
| **按键** | 3× 用户按键 + 1× 电源键 |
| **待机功耗** | 92.53 μA |
| **满载功耗** | 211.97 mA |
| **尺寸** | 70.8 × 103.9 × 8.5 mm |
| **重量** | 73.3 g |

## 电池特性

| 参数 | 值 |
|------|-----|
| 电池类型 | 锂电池 |
| 容量 | 1250 mAh |
| 满电电压 | 4.2V |
| 截止电压 | 3.0V |
| 低压关机 | 3.1V (M5PM1 检测) |
| 充电 | USB 5V，M5PM1 管理 |

## 显示参数

| 参数 | 值 |
|------|-----|
| 面板 | E Ink Spectra 6 |
| 型号 | ED2208-DOA (EL040EF1) |
| 尺寸 | 4 英寸 |
| 分辨率 | 400 × 600 |
| 色深 | 8-bit (全彩) |
| 刷新模式 | epd_quality (高质量), epd_fastest (快速) |
| 驱动 | M5GFX + 离屏 Canvas (PSRAM) |
