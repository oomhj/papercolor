# 连接生命周期管理

WiFi 连接的自动重试、失败处理、手动触发配网。

---

## 一、连接状态机

```
         ┌──────────┐
         │  INIT    │  ← 系统启动
         └────┬─────┘
              │
         ┌────▼─────┐
         │ LOAD_NVS │  读取保存的 WiFi 配置
         └────┬─────┘
              │
         ┌────▼─────┐
         │ STA_RETRY│  尝试连接（最多 3 次）
         └────┬─────┘
              │
        ┌─────┴──────┐
        │            │
    ┌───▼──┐    ┌───▼────┐
    │ ONLINE│    │AP_PROV │  ← 全部失败，进入配网
    └───┬──┘    └───┬────┘
        │           │
    ┌───▼──┐    ┌───▼────┐
    │ LOST  │    │ CFG_DONE│← 用户提交 WiFi 配置
    └───┬──┘    └───┬────┘
        │           │
        └── 重试 ───┘
        （与 STA_RETRY 相同）
```

---

## 二、连接策略

### 2.1 首次连接

```
尝试第 1 次：超时 10s
  ├─ 成功 → ONLINE
  └─ 失败 → 间隔 2s

尝试第 2 次：超时 10s
  ├─ 成功 → ONLINE
  └─ 失败 → 间隔 5s

尝试第 3 次：超时 15s
  ├─ 成功 → ONLINE
  └─ 全部失败 → 启动 AP 配网
```

### 2.2 运行中断线重连

正常使用中 WiFi 断线：

```
断线检测 → 状态变为 LOST
  ├─ 第 1 次重试：立即
  │   ├─ 成功 → ONLINE
  │   └─ 失败 → 等待 5s
  ├─ 第 2 次重试：等待 5s
  │   ├─ 成功 → ONLINE
  │   └─ 失败 → 等待 15s
  ├─ 第 3 次重试：等待 15s
  │   ├─ 成功 → ONLINE
  │   └─ 全部失败 → 启动 AP 配网
```

断线重试不会无限进行，最多 3 次后进入 AP 模式。

### 2.3 重试间隔

| 尝试次数 | 等待时间 | 原因 |
|----------|----------|------|
| 1 | 0s（立即） | 可能是短暂信号波动 |
| 2 | 5s | 路由器重启中 |
| 3 | 15s | 网络问题，给时间恢复 |
| 之后 | 进入 AP 配网 | 不再自动重试 |

## 三、手动触发配网

用户在运行中可通过按键主动进入配网模式：

### 操作方式

| 操作 | 当前状态 | 结果 |
|------|----------|------|
| **长按 BTN-UP 3 秒** | STA 已连接 | 启动 AP + 配网页，原 STA 保持 |
| **长按 BTN-UP 3 秒** | STA 断线 | 启动 AP + 配网页 |
| **长按 BTN-UP 3 秒** | AP 模式中 | 刷新配网页（重新扫描） |

### 按键定义

| 按键 | GPIO | 功能约定 |
|------|------|----------|
| BTN-TOP | G1 | 顶部按键（长按 5s 休眠） |
| BTN-UP | G9 | 上/前一页 / 配网（长按 3s 进入配网） |
| BTN-DOWN | G10 | 下/后一页 |

### LED 反馈

| 操作 | LED 效果 |
|------|----------|
| 按下 BTN-A | 蓝灯亮 |
| 3 秒到达 | 绿灯闪烁一次，AP 启动 |
| AP 已启动 | 蓝灯快闪 |

---

## 四、配网完成后的行为

用户通过配网页提交 WiFi 配置后：

```
1. 保存到 NVS (slot 0)
2. 返回 HTTP 200 {"status":"ok"}
3. 延时 500ms（让浏览器收到响应）
4. 重启设备（esp_restart()）

重启后：
  ├─ 加载 NVS → 有配置
  ├─ 尝试连接 → 成功 → ONLINE
  └─ 尝试连接 → 失败 → 再次进入 AP 配网（用户可检查密码）
```

---

## 五、API 扩展

在 `wifi_manager.h` 中增加：

```c
/** @brief 手动触发配网模式（由按键调用） */
void wifi_mgr_trigger_provisioning(void);

/** @brief 开始连接重试循环（含退避，最多 3 次） */
void wifi_mgr_start_retry_loop(void);

/** @brief 停止重试 */
void wifi_mgr_stop_retry(void);

/** @brief 重启设备 */
void wifi_mgr_restart(void);
```

## 六、LED 状态总表

| 状态 | LED | 操作 |
|------|-----|------|
| INIT | 熄灭 | 系统启动中 |
| STA 连接中 (第1次) | 蓝灯常亮 | 等待 10s |
| STA 重试 (第2次) | 蓝灯慢闪 (1s) | 等待 5s 后重试 |
| STA 重试 (第3次) | 蓝灯快闪 (300ms) | 等待 15s 后重试 |
| STA 已连接 (ONLINE) | 绿灯 0.5s 后熄灭 | 正常工作 |
| STA 断线 | 红灯 1s | 进入重试 |
| AP 配网中 | 蓝灯交替快慢闪 | 等待用户配置 |
| 配网完成 | 绿灯闪烁 3 次 | 即将重启 |
| 休眠 | 熄灭 | 深度睡眠 |
